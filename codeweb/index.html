HavokMud Resources Center

Welcome to Havok's Mud resource center. My name is Greg Hovey and this is my little page devoted mainly to HavokMud which is a Multi User Dungeon. Havok currently uses the great DaleMud code base. The purpose of this site is to set up some of my own code snippets and other mud resources for all to see. This site was originally suppose to be a devolvement site for Havok but slowly  became obsolete since our convergence into Sourceforge.  Despite that, I'll try to come up with some other resources.
