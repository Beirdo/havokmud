HAVOKMUD 1.9.3 News File

.:*~*:._.:*~*:._.:*~*:._.:*~*December 24th, 2002*~*:._.:*~*:._.:*~*:._.:*~*:.

The New Stuff
-------------
-]New Features[-
------------
-Weapons speeds are now implemented. (All defaulted to +0.25) 
-States speed on ID
-New Battle prompt. (Set with the 'bprompt' command)
-MOTD supports color codes now
-When gaining levels, now states how many hp, mana, and Movement you gained.
-New Auction system (Not working yet, been disabled)
-Money can be dropped/given/deposited/withdrawn by using the k or m letter.
-example: 'give 56k to banon' would give me 56000 coins.
-A new Experience Cap has been implemented. (10% of the exp. needed to level)
-New sector types implemented. (adds negative and positive regen accordingly)
-Regen Rooms
-Fire sector
-Electricity sectors
-etc
-Spell experience is based on the mana used for the spell.
-xp=manacost*50 
-will also get filtered through new xp cap
-New 'set' options. usage: set autoXXXX <disable/enable>
-Autosplit, will automatically split any gold that you pick up.
-Autogold, automatically get the gold out of a corpse
-Autoloot, automatically loot all the items out of a corpse.
-Autoassist, will automatically assist your group leader in battle.
-New 'resist' command. States all character resistances.
-New number formatter for large numbers. (ie. 23,119,999)
-New Fighting Styles. Set with the 'style' or 'fight' command.
-Standard
-Aggressive, +dam and decreases your armour
-Defensive, -dam and increases armour
-Evasive, -number of attacks and increases your AC and chance to flee
-Berserked, +Number of attacks and decreases AC.

-]Misc[-
----
-Monsume monsters have more Hitpoints.
-Ability to pick alignment on character creation.
-Troll, Goblin, orc, and dark darf are choosable on character creation.
-Keyword 'study' works the same as 'practice' now. (For you SD people)
-'C' is short for 'cast' now, not cackle.
-Color added to 'equip' command.
-New ID command. Basically an alias for cast 'spell' name.
-Usage: 'ID <Itemname>'
-Trolls regen better along with regen message.


-]Bugs[-
----
-'Draw' command can be used during battle now.
- Monks 'legsweep' skill can be used during battle also.
- Few Crashed bugs fixxed.


-]Newbie Stuff[-
------------
-'Help' command is more user friendly.(Smarter looking for keywords)
-Newbie advice channel. 
-Gives newbies random help messages.
-Can activated and deactivated with the set command.'set advice <dis/en>
-'Who' command states if a character is new or not.
-Will say how much xp received from a kill when ungrouped.


-]Immortal Stuff[-
--------------
-New pretty 'Wizhelp' command.
-Some new crash bugs fixxed.
-Rare has its own flag now.
-Rare field pickable on oedit.
-Rent is the actual price now. No need to divide number by 2.
-Can set object speed with the 'setobjspeed' command.
-Max manager. can see list of maxxed items.
-Usage: 'show max <name of object|zone#>'
