Newbie Guide: Talking
 

There's basically four primary modes of communication in the world of Havok.  There's world-wide chat, which everyone in the game can hear; there's zone-wide chat, which everyone in the same zone as you can hear; there's room-chat, which everyone in the same room as you can hear; and there's private chat, which only you and select others can hear.

World-wide chat
The two world-wide communication commands are shout and ooc.  You can't use either of these until your character is Level 2.  Ooc is short for Out Of Character, and you should use this for talking about non-game related things.  Shout is for character-related discussions, like roleplaying.  Keep in mind that you shouldn't keep a constant stream of spam flying across the world - not everyone might want to hear what you have to say all the time.

You can combine socials and emotes with the ooc command.  (Type help social for more info on these.)  To do a social, type OOC %(social) (target), and to do an emote, type OOC #(whatever you want to emote).

Zone-wide chat
The only zone communcation command is yell.  (Note that all Immortal Staff members can hear all yells.)  Everyone on the same zone as you can hear yells.  Type whozone to see who's in the zone with you.

Room chat
The main room chat command is say.  It's used like this: say Wow mom, look at me, I'm talking.  But there's also emote, which is good for roleplaying, and is used like: emote is doing something crazy behind your back. 

Private chat
The main private chat command is tell.  You can only tell to someone that's in the same zone as you, although you can tell to any Immortal at any time.  Certain classes have spells such as telepathy and sending which allow them to talk to a specific person anywhere in the game.

Another one is group tell, which is used as: gt Man, I'm really glad we're in this group together, guys!  Only people in your group can hear group tells.  See the chapter on grouping to learn more on that.
