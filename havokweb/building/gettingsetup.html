Getting Set Up

  For a complete walk-through, check out the Guide to Building on Havok: these pages are intended as quick-references.

 Getting set up is the easiest part: Simply contact myself (or another Implementor, preferably someone who's familliar with the current state of affairs on the buildsite), and tell them that you need to get set up.  Log on to the buildsite, create a new character (with the same name as your playsite Immortal, if you have one), and we'll do the rest.
