Coders Corner

In the past few years, Havok has undergone some very critical changes and alterations.  To see a list of some of these changes from the last 2 years, then I suggest visiting our HavokMud resource center where other coding resources such as snippets and links can be found.  A starter guide to coding can also be found there. 

Unfornately this web page is no longer supported and maintained.  This was due to Havok going open source on SourceForge which was a very good thing.  Especially for prospective mud coders or people looking to startup there very own mud using our wonderful DaleMUD based source code.  So if you are curious about our code then feel free to download our it on our SourceForge web page.
